1 – Crie uma aplicação OpenGL para o Jogo “Atravessando o Labirinto”, conforme
ilustração a seguir:

![](https://i.imgur.com/qMQzpTq.jpg)

O objetivo é passar um objeto (circulo, triângulo ou retângulo) sem tocar nas paredes do labirinto. Cada vez que ocorrer um toque na parede, o objeto volta para o início e o jogador perde uma “vida”. O jogo acaba quando ele atravessar sem tocar (vitória) ou bate 4 vezes em uma parede (derrota).

Outras características:</br>
1 - O objeto deve estar preenchido com uma cor de escolha da dupla;</br>
2 - A largura do labirinto deve ser 1,5 vezes (aproximadamente) o tamanho do objeto;</br>
3 - Utilizar as setas do teclado para movimento do objeto no labirinto.</br>
4 - O botão esquerdo do mouse deve ser usado para mudar as cores da janela, do labirinto e do objeto.</br>
5 - Outros requisitos para o funcionamento do jogo (dimensões, cores, formato do labirinto etc..) podem ser definidos pela dupla.</br>
